# Library Mode for NeMo Retriever Library

!!! note

    NVIDIA Ingest (nv-ingest) has been renamed to the NeMo Retriever Library.

Use the [Quick Start for NeMo Retriever Library](ttps://github.com/randerzander/NeMo-Retriever/blob/main/nemo_retriever/README.md) to set up and run the NeMo Retriever Library in local library mode, so you can build a GPU‑accelerated, multimodal RAG ingestion pipeline that parses PDFs, HTML, text, audio, and video into LanceDB vector embeddings, integrates with Nemotron RAG models (locally or via NIM endpoints), which includes Ray‑based scaling plus built‑in recall evaluation.
